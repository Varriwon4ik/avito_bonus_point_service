Autotester

Runs the built-in accrual autotester against this live instance — the same checks as the cmd/autotest console tool. It verifies accrual correctness and, crucially, that a burst of parallel accrual requests each produce a distinct lot with a consistent balance and ledger. All requests use a dedicated autotest- test user, so your real accounts are never touched.
